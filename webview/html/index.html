Herzlich Willkommen zum FutureC Designer! (Erstentwurf)

Dies ist die Hilfe, die das erste mal geöffnet wird, sobald man den Designer öffnet. Später kann man diesen Dialog über das Fragezeichen oben rechts öffnen. Um sich die verfügbaren Keybindings erneut anzuschauen.

Keybindings:

Ctrl + L-Maus + Element: Öffnet einen Editdialog mit der Möglichkeit mehrere Eigenschaften eines Elements anzupassen

Ctrl + ⇧ Shift + Alt + L-Maus + Element: Entfernt ein Element aus dem Designer und aus dem Source-File

Ctrl + Alt + L-Maus: Erzeugt ein neues Element. Danach wird man wird nach der Spaltennummer abgefragt um diese in das neu erzeugte Element zu speichern.

Neu seit letztem Update (akt.: VERSION_NUMBER):

Es gibt die Möglichkeit mehrere Elemente per Drag & Drop zu markieren und sie zu verschieben.


ACHTUNG!!!
Das löschen eines Elements blendet es nicht aus. Um ein Element aus dem Dialog auszublenden, muss man das Element auf VISIBLE=0 setzen
